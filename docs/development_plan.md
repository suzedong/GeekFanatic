## 开发指南

### 命名规范

- 类名：PascalCase
- 方法名：snake_case
- 私有成员：_前缀
- 常量：UPPER_CASE

### 文件组织

```
src/geek_fanatic/
├── core/           # 核心功能
├── plugins/        # 插件目录
└── resources/      # 资源文件
```

### 代码规范

- 类型注解
- 文档字符串
- 单元测试
- 代码格式化

### 提交规范

- feat: 新功能
- fix: 修复问题
- docs: 文档更新
- style: 代码格式
- refactor: 代码重构
- test: 测试相关
- chore: 构建过程
